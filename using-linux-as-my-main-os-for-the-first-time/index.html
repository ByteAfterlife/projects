Using Linux as my Main OS for the first time

Byte November 29, 2025

Ever since August 2024, I've been using Linux as my main OS. Why? Well it's more private, they don't install unnecessary apps like other OS's, and you can customize it as much as you want. Simple. I also like it because lots of servers use it and in the future if I administer servers, learning linux is good, so far I know alot, package managers, software, fun easter eggs, etc. I like it, I also like something called "LUKS" which is basically a encryption system that encrypts your drive data, I'm not interested in that part, I like the way it stores the encryption key.. Instead of your password being the encryption key, the encryption key is actually a random string from what I know, and THAT string is encrypted with your password, so your password is used to get that encryption key to decrypt your data. But with that.. You can setup multiple "slots" or holders of that encrypted encryption key copy.. Like a security key could hold the same encryption key, and decrypt your data. I find it a creative idea!

Also, as I said it's really customizable. I added a notepad widget to my screen, a extension to make my screen more red, and way more that's not included in this screenshot, but here's my desktop.
