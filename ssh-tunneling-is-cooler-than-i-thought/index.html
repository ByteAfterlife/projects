SSH tunneling is cooler than I thought

Byte December 20, 2025

I've known about SSH tunnelling for a long time, I never actually know how it works though. Now I learned how it actually works and it's so much better. Basically how it works is if you want to access things on another computer that are locked down, you can use a program called SSH to make a tunnel into that computer and access the things on it, and it also binds to a port, which is kind of like a room in your computer, and forwards it to another room in the other computer, so lets say on port 8080 on your remote computer, you're hosting a secret website, but it can't be accessed through the internet for whatever reason, or doesn't have HTTPS. You can make an SSH tunnel into the computer, and as long as SSH can be accessed from anywhere, anything on that computer can be accessed from anywhere. It's pretty cool. It also encrypts stuff so if your program can't encrypt connections when needed, like no HTTPS, SSH does it for you.
