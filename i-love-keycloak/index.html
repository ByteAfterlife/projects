I love Keycloak

Byte November 29, 2025

Ever since a year ago I've been experimenting with keycloak and oh my it is beautiful!

You can setup SAML and OIDC clients so easily, and they even integrate with LDAP and Kerberos. One time I used Keycloak is with my inventory tracker, I had some other family related apps, and I got them to use Keycloak to sign in with one password into all of them. It made it way more convenient for them. I also of course used it with docker because I was then and still am currently obsessed with docker.

Unlike other open source identity providers like OpenLDAP or Authentik or some others like that, Keycloak is easy to use. I just tell you, it took me an hour entering ldap commands into openldap just to get one more attribute into a user, their fingerprint ID for a system I was making, and I still couldn't get it. In keycloak, it was way easier. I just did Realm Settings > User profile > Create attribute, filled in the fields, and I'm done! I also learned how to retrieve those fields in their API, so in my system it queried the API using a "service account" client, it searched for a user based on the fingerprint ID, and found the user's details. In my opinion, if you need a good balance between customizability and simplicity, use Keycloak.

Have fun!
